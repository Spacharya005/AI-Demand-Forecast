# ☕ AI Demand Forecasting Dashboard

A data-driven forecasting system designed to predict beverage demand and support proactive decision-making for coffee businesses. This project integrates machine learning, time-series analysis, and interactive visualization using Streamlit and FastAPI.

---

## 🚀 Project Overview

This project aims to transform **reactive decision-making into proactive planning** by analyzing historical sales data and predicting future demand patterns.

It provides:

* 📈 Accurate demand forecasting
* 🧠 Automated business insights
* 🚨 Anomaly detection
* 📊 Interactive dashboard for visualization

---

## 🧩 Features

### 🔹 Overview Dashboard

* API health monitoring
* Data preview and validation
* Hourly demand distribution visualization

### 🔹 Forecasting Module

* Time-series prediction using ML models
* Rolling average trend analysis
* Interactive Plotly visualizations

### 🔹 Insights Engine

* Automatically generated business insights
* Data-driven recommendations for operations

### 🔹 Anomaly Detection

* Detects unusual spikes or drops in demand
* Helps identify operational or seasonal issues

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** FastAPI
* **Data Processing:** Pandas, NumPy
* **Visualization:** Plotly
* **Machine Learning:** Scikit-learn, Time-Series Models (ARIMA / Prophet)
* **Version Control:** Git, GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/ai-demand-forecasting.git
cd ai-demand-forecasting
```

### 2️⃣ Create a virtual environment

```bash
python -m venv venv
source venv/bin/activate   # For Linux/Mac
venv\Scripts\activate      # For Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

### 🔹 Start FastAPI Backend

```bash
uvicorn api.main:app --reload
```

### 🔹 Start Streamlit Frontend

```bash
streamlit run app/app.py
```
